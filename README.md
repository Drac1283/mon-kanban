## Installation du projet

Cloner le projet

```bash
  git clone https://github.com/Drac1283/mon-kanban.git
```

Aller dans le répertoire

```bash
  cd mon-kanban
```

Installer les dépendance

```bash
  npm install
  npm install -g vercel
```

démarrer le serveur

```bash
  vercel dev
```
